# Code of Conduct

## Our standard

We expect all participants to communicate professionally and respectfully.

## Expected behavior

- Be constructive and specific in technical discussions.
- Assume good intent and focus on the code and data.
- Respect different levels of Bloomberg/domain experience.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Publishing private or sensitive information.
- Disruptive, bad-faith interactions.

## Enforcement

Project maintainers may remove content or restrict participation for behavior that violates this code.

## Reporting

Report issues to the maintainer privately via the contact path in `SECURITY.md`.
